# Sahid AI Hub

The public command center for everything Sahid Attaf builds with AI.

Live applications, open-source repositories, and proof-of-work prototypes spanning hospitality, commerce, workforce intelligence, and platform engineering.

## Stack

- **Framework:** Next.js 16, App Router, TypeScript
- **Styling:** Tailwind CSS v4
- **Deployment:** Vercel

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run lint
```
